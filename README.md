# Premium Developer Portfolio (React + Vite)

A modern, responsive developer portfolio with dark gradients, glassmorphism effects, GSAP animations, and Lenis smooth scrolling.

## Run locally

```bash
npm install
npm run dev
```

Then open the Vite URL shown in the terminal (usually `http://localhost:5173`).

## Build for production

```bash
npm run build
npm run preview
```
